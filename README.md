# AI-Powered E-Commerce Customer Review Intelligence Agent

## Overview
This project analyzes customer reviews from e-commerce platforms using AI and NLP techniques.

## Features
- Sentiment Analysis
- Review Classification
- Dashboard Visualization
- Complaint Area Detection
- Actionable Insights

## Technologies Used
- Python
- Streamlit
- Pandas
- NLP
- Machine Learning

## Dataset
E-commerce electronics review dataset.

## How to Run
1. Install dependencies
2. Run:
   streamlit run app.py
